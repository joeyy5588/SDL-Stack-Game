# SDL-Stack-Game

## Introduction
A game implemented with C++ and SDL library. Press SPACE to change direction, and try not to pass the checkpoint line with specific combination, or you might fail! Have fun!

## Usage
```
$ cd bin/Debug
$ Execute the exe file
$ Hvae Fun!
```
